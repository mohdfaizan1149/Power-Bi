# Power-Bi
# 🍕 Pizza Sales Analysis Dashboard – Power BI Project
# 📌 Project Overview
This project is a comprehensive Pizza Sales Analysis Dashboard built using Power BI Desktop to analyze sales performance, customer ordering patterns, product categories, and time-based trends.
The dashboard transforms raw sales data into meaningful business insights that help understand customer behavior, peak sales periods, and category performance.
# 🎯 Objective
The main goal of this project is to:
Analyze total sales performance
Identify peak order times and busiest days
Compare monthly sales trends
Evaluate category-wise sales contribution
Support data-driven business decisions
# 📊 Key Insights
Total Sales: 817.9K
Total Orders: 21K
Total Quantity Sold: 50K
Top Performing Category: Classic Pizza
Busiest Day: Friday
Highest Sales Month: July
# 📈 Dashboard Features
Sales performance overview
Orders by day of the week
Orders by custom time intervals
Monthly sales trend analysis
Category-wise sales distribution
Product performance comparison
# 🛠 Tools & Technologies Used
Power BI Desktop
Data Cleaning & Transformation
DAX (Data Analysis Expressions)
Interactive Visualizations
Business Intelligence Reporting
# 💡 Business Value
This dashboard helps businesses:
Optimize inventory planning
Improve sales strategies
Understand customer preferences
Maximize revenue through trend analysis
# 🚀 Conclusion
This project demonstrates how data visualization and business intelligence tools can convert sales data into actionable insights, enabling smarter business decisions and operational efficiency.
